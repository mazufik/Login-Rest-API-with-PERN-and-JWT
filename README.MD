# Login REST API with PERN Stack and JWT (Backend)

simple Rest API for login using Node JS, Express, JWT and PostgreSQL.

### Feature

1. Login
2. Logout
3. Token
4. Show Users
5. Create User

### Installation

```
GIT CLONE https://github.com/mazufik/Login-Rest-API-with-PERN-and-JWT.git
```

```
NPM INSTALL
```

```
NPM START
```
